## 개요

SEDP는 오픈소스 담배 연기 감지 시스템입니다.

전국의 어떤 공학 동아리에서든지 쉽게 따라 만들고 설치하여 교내 흡연 문제를 종식시키는 것을 목적으로 하고 있습니다.
